## Project Description
The **CyberSecurity ChatBot** is a console-based C# application designed to educate users about cybersecurity best practices. The bot interacts with users, providing guidance on topics such as password safety, phishing, safe browsing, malware, social engineering, and device security. The project demonstrates practical use of C# 10 with .NET 6 and implements a clean, user-friendly command-line interface.

---

## Features
- Personalized greeting with user name.
- Provides advice on cybersecurity topics:
  - Password safety
  - Phishing awareness
  - Safe browsing tips
  - Malware protection
  - Social engineering prevention
  - Device security
- Exit the chatbot anytime with the `exit` command.
- Typing effect to simulate a real chat experience.
